## FS26 Cohort Course Plan
Official FS basis-year setup from D-MATH degree details:
- Semesters 1-2 together form the **Basisjahr** (60 KP total target).
- **Compulsory subjects:** 50 KP.
- In semester 2, choose **one supplementary subject**:
  - Physics II (5 KP), or
  - Data Structures and Algorithms (8 KP).

### Analysis II: Several Variables (401-1262-07L)
- Course overview:
  - Multivariable differential and integral calculus, with proof-based foundations.
- Official page/link:
  - VVZ: https://www.vorlesungen.ethz.ch/Vorlesungsverzeichnis/lerneinheit.view?lang=en&semkez=2026S&ansicht=LEHRVERANSTALTUNGEN&lerneinheitId=193395
  - D-MATH course site hub: https://metaphor.ethz.ch/x/2025/fs/401-1262-07L/
- Weekly structure (week-by-week checklist):
  - Week 1: set up script template, compile notation sheet, attend first exercise.
  - Week 2: complete first full problem set without skipping proof steps.
  - Week 3: create theorem dependency map (implicit function theorem chain).
  - Week 4: do one timed 45-minute exercise block.
  - Week 5: consolidate coordinate transforms + Jacobian problems.
  - Week 6: mid-semester recap sheet (max 2 pages).
  - Week 7: practice line/surface integral setups from scratch.
  - Week 8: run one past-problem set under 90-minute timing.
  - Week 9: patch weak spots from Week 8 error log.
  - Week 10: complete all open exercise backlogs.
  - Week 11: mixed-topic drill (differentiation + integration + geometry).
  - Week 12: second timed mock (120 minutes).
  - Week 13: oral explanation drills with peer (5 key theorems).
  - Week 14: final formula/theorem compression and error checklist.
- Assessment format/date:
  - Written end-semester examination, **180 minutes**.
  - Exact date: `TODO` (publish via myStudies/department schedule).
- Materials and deliverables:
  - Lecture notes, exercise sheets, personal theorem map, timed mock logs.
- What to do this week:
  - [ ] Attend lecture + exercise session.
  - [ ] Finish and self-correct weekly problem set.
  - [ ] Add new theorems and common errors to your running sheet.

## Basisprufung and Exam Blocks
- Verified from D-MATH degree details and the official bachelor guide:
  - Basisjahr spans semesters 1-2.
  - Compulsory subjects total 50 KP.
  - Examinations are organized in blocks.
- Block mapping relevant for FS26:
  - **Block 1:** Analysis I, Analysis II, Linear Algebra I, Linear Algebra II.
  - **Block 2:** Introduction to Programming, Basic Structures in Discrete Mathematics.
- Caveat:
  - Exact exam dates, rooms, and per-student allocations are not published here; check myStudies and D-MATH communication channels when released.
